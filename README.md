# Sistem Informasi Tindak Lanjut Rekomendasi BPK

### Teknologi 

* Laravael 10
* Bootstrap 5 
* Mazer Template
* MySQL

### Abstract
```
Badan Pusat Statistik (BPS) menyusun laporan keuangan sebagai pertanggungjawaban anggaran dan menyerahkannya 
kepada Badan Pemeriksa Keuangan (BPK) untuk diperiksa. Hasil pemeriksaan BPK disajikan dalam laporan hasil 
pemeriksaan (LHP). Inspektorat Utama mengarahkan pelaksanaan tindak lanjut rekomendasi BPK jika ada temuan dalam 
LHP. Saat ini, terdapat sistem dashboard monitoring progres dan hasil tindak lanjut rekomendasi BPK. Namun, sistem ini 
tidak mencakup seluruh proses bisnis tindak lanjut rekomendasi, memiliki keterbatasan dalam memberikan akses yang tepat, 
serta tidak adanya penyimpanan dokumentasi yang terpusat. Oleh karena itu, dibutuhkan sebuah sistem informasi untuk 
mengatasi masalah tersebut. Sistem dibangun menggunakan metode iterative dan dievaluasi dengan black box testing serta 
Computer System Usability Questionnaire (CSUQ). Hasil evaluasi black box testing menunjukkan bahwa sistem berjalan 
sesuai dengan fungsionalitas yang diharapkan. Selain itu, hasil evaluasi CSUQ menunjukkan bahwa pengguna sangat puas 
dengan kegunaan, kualitas informasi, dan antarmuka sistem.
```
